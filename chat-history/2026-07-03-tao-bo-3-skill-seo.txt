  3️⃣ backlink-analyzer — phân tích backlink từ Ahrefs

  Đầu vào: người dùng gửi file Excel dữ liệu Referring Domains export từ Ahrefs
  của một hoặc nhiều website cần phân tích (so sánh mình vs đối thủ).

  Quy trình (steps trong SKILL.md):
  1. Đọc file Excel → nhận diện các cột Ahrefs chuẩn (Domain, DR, Domain
  Traffic, Referring Domains, Linked Domains, Links to Target, First Seen, Last
  Seen, Link Type: dofollow/nofollow...).
  2. Làm sạch dữ liệu (bỏ trùng, chuẩn hóa domain).
  3. Phân tích:
    - Phân bố DR (chất lượng ref domain: cao/trung/thấp)
    - Tỷ lệ dofollow / nofollow
    - Top ref domains theo DR & traffic
    - Link gap — domain trỏ về đối thủ nhưng chưa trỏ về mình (cơ hội xây link)
    - Link mới / mất theo First Seen – Last Seen
    - Cảnh báo backlink độc/spam (DR thấp + traffic 0)
  4. Xuất báo cáo .xlsx (nhiều sheet) + tóm tắt .md, biểu đồ nếu cần → lưu
  outputs/.

  File bổ trợ (load-on-demand):
  - config/ahref-columns.md — ánh xạ tên cột Ahrefs (phòng khi export tiếng
  Anh/Việt hoặc đổi thứ tự).
  - rules/backlink-quality-rules.md — ngưỡng đánh giá (DR ≥ 30 = tốt, traffic =
  0 nghi spam, tỷ lệ dofollow lý tưởng...).
  - templates/backlink-report-template.md — khung báo cáo.

  Ăn điểm: đây là skill thứ 2 có folder bổ trợ (củng cố điểm 1.4), có đủ steps +
  error handling (file sai định dạng, thiếu cột) + tiêu chí chất lượng + format
  output.
